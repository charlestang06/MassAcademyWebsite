RFID Sign-in/out System

In B term, Tarun Eswar, David Barsoum, and I worked on our independent project to improve the sign-in/out system at MAMS. The previous sign-in system at Mass Academy was very mechanical and time-consuming. We developed this project to improve the efficiency and technology related to the sign-in and sign-out methods. This digital sign-in/out system uses SQL databases, ReactJS web portal, NodeJS backend, and a Raspberry Pi device for interacting with RFID tags.
